## U+3099／U+309A（結合濁点・結合半濁点）

結合文字にも対応しています。例として **ゔ** を入力する場合：

1. **う** と入力して確定
2. **3099** と入力
3. **F5** でUnicode文字へ変換して確定

これで **う** の後ろにU+3099が入り、**ゔ** になります。半濁点は **309A** を同じ手順で入力します。

日本語IMEを使わない場合は、**ゔ** をそのままコピー＆ペーストするか、結合濁点 **゙** をかなの後ろに貼り付けてください。Word／Officeでは **3099 → Alt+X** でも入力できます。表示結果はアプリの結合文字対応に依存します。

## English

GEKIFUDE-SFX is a free display font for manga sound effects, Japanese onomatopoeia, impact lettering, and speech-bubble text.

It uses fast, expressive brush movement, heavy entry strokes, sharp finishes, an upper-weighted shape, and a strong taper toward the bottom. Created by **HIDEOUT**.

### Combining marks: U+3099 / U+309A

To type **ゔ** with a Japanese IME, confirm **う**, enter **3099**, press **F5**, and confirm the converted character. For users without a Japanese IME, copy and paste **ゔ** or paste the combining mark **゙** after a kana. In Word／Office, **3099 → Alt+X** also works. Use **309A** for a handakuten.
