# GEKIFUDE-SFX
<img width="3200" height="964" alt="download_3" src="https://github.com/user-attachments/assets/c8ae7262-ab7e-418d-a2a5-25e2103acbc2" />


漫画・オノマトペ／SFX向けの筆文字フォントです。作者は HIDEOUT。
文字の勢いと上重心・下すぼまりのシルエットを重視しています。

## ダウンロード

- [GEKIFUDE_SFX_v1.002_CC0.zip](./GEKIFUDE_SFX_v1.002_CC0.zip)
- [使用可能文字一覧（横長PNG）](./GEKIFUDE_SFX_glyphs_horizontal.png)

ひらがな・カタカナ、濁点／半濁点、U+3099／U+309A、英大文字、ASCII記号、全角数字、ハート・矢印・チェック、漫画向け感情／スピーチバブル記号を収録しています。
